📥 Export Data
Download your data as CSV files — open directly in Excel or Google Sheets. Connect Power BI using the JSON feed URL.
📊 How to Connect Power BI
1. Open Power BI Desktop → Get Data → Web
2. Paste your Power BI URL (click button above)
3. Click OK → Power BI loads all your farm data
4. Build charts for diseases, languages, regions
5. Refresh anytime to get latest data
📗 How to Open in Excel
1. Click "Download Diagnoses CSV" above
2. Open Excel → File → Open → select the downloaded CSV
3. All diagnoses appear as rows with columns for disease, category, language, date
4. Use Excel pivot tables to analyse by region, disease, or language
